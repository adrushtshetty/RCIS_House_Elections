Installation

The following includes the process to be followed to prepare the platform for a new election.


Individual System

  1. There are two methods for hosting this platform. The first method involves utilizing the application on individual systems, where each system must be equipped with the required modules listed in the "requirements.txt" file located in the application's root folder.

Server Version

  1. The second method entails utilizing a single system as the host, with all devices connected to the network being able to access the application using the IP address for the host computer over the network. The link to access the application can be found in the PyCharm terminal.

    Utilities

    This section comprises the descriptions and functionalities of several tools available on our platform to assist users in conducting elections.

    Accessing Utilites

    The utilities described below can be located by the user in the administrative dashboard.

    Dashboard

    Database Cleaner

    This tool permits the user to erase the databases utilized for vote counting and initiate a new election. Exercise caution when accessing the result page after clearing the databases, as it will result in an error

    Dashboard


    Upon clicking the "submit" button, the user can view the status and verify that the databases have been successfully cleared.

    Login Credentials

    This tool, located in the navigation bar, enables the user to modify their login credentials, view their current login information, and edit the recovery email addresses to which their credentials will be sent.

    Nav Bar




    Upon inputting their password and username, the user can verify their status on the right-hand side.



    The user can access their current login credentials by selecting the button below.



    By selecting the "MAIL ID" button, the user can add their email addresses to the list of recipients who will receive login credentials in the event that they are forgotten.

    Clearing Candidate Details

    This tool enables you to eliminate candidate information from the entire application, thereby providing you with the ability to enter details for subsequent candidates.

    Nav Bar




    Upon selecting the radio button for clearing the database and submitting the form, it is anticipated that the status will indicate "Candidates Details Cleared."
